# PACOTE PÚBLICO — SEGURO PARA GITHUB

Este pacote contém somente o código do aplicativo.

## Não publicar

Nunca coloque no repositório:

- a pasta `migration/`;
- o pacote privado de migração;
- PDFs, fotos, planilhas ou documentos de clientes;
- `Marco_Iris_Dados.migrado.json`.

A migração histórica é feita selecionando no navegador uma pasta privada armazenada no computador. O aplicativo lê esses arquivos localmente e envia as mídias diretamente ao Google Drive conectado.


Na tela Configurações → Backup e migração, use o cartão “Migração histórica preparada”. Não use a migração genérica do AppSheet para este pacote.
