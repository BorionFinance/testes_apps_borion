# LEIA PRIMEIRO — Marco Iris v2.6.0

Este pacote substitui a v2.5.9.

## Como publicar

1. Extraia o arquivo ZIP.
2. Abra a pasta extraída da v2.6.0.
3. Envie para o repositório do GitHub **os arquivos que estão dentro dessa pasta**.
4. Não publique o ZIP como um único arquivo e não crie uma pasta extra acima do `index.html`.
5. Aguarde o GitHub Pages concluir a implantação.
6. Na primeira abertura, use `Ctrl + F5` no computador para garantir a troca imediata da versão antiga.

## Conferência rápida depois da publicação

- Abra a Visão Geral e confira os botões do Faturamento na ordem **Ano, Mês, Dia**.
- Abra Nova OSV, Novo Cliente e um lançamento financeiro.
- Clique em **Editar layout**, mova um bloco, salve, feche e abra novamente.
- Confirme que o bloco permanece na posição e no tamanho salvos.

Os dados operacionais, PDFs e integrações existentes não foram removidos. A atualização atua no editor visual, persistência dos layouts e no gráfico de faturamento.
